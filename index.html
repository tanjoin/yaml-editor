YAML Editor 機能ガイド

リアルタイム検証

入力されたYAMLをリアルタイムで構文チェックします。エラーを検出した場合はステータスバーとプレビュー領域にエラー行や詳細を明示します。

ツリー構造プレビュー

YAMLの親子構造をビジュアルツリーとして綺麗にマッピング。トグルをクリックしてオブジェクトや配列を手軽に折りたたむことが可能です。

各種形式へのフォーマット変換

ワンクリックでJSON・XML・TOMLデータへと変換します。数値を不必要に省略したり丸めたりせず、正確な数値をそのまま維持してコンバートします。

スマートTabインデント

Tab で2スペースを挿入し、Shift + Tab でインデントを戻せます。複数行を一括選択した状態でのスマートインデントにも対応しています。

改行自動インデント（スマートアシスト）

改行（Enter）時に直前の行のインデントを自動算出。さらに行末が : または - で終わる場合は自動的にネストを1段階（+2スペース分）深く調整します。

入力キーワード補完 & 自動補完ペア

YAMLの主要キーワードを英単語の入力中に予測表示（矢印キーで選択しEnter/Tabで決定）。また、" ' { [ を打ち込んだ際には右側の閉じ文字も自動挿入します。

自動保存＆ローカル復元

エディターでの入力は随時ブラウザの `localStorage` へとオートセーブ。ページを誤ってリロードしたり一度閉じた場合でも、次回起動時に前回の作業を即座に復元して作業が再開できます。
